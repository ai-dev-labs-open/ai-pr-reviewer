# Roadmap

## Near Term

- Inline review comments for findings that map cleanly to diff hunks
- Optional repo-level config file for severity thresholds and custom instructions
- Better binary-file and generated-file detection
- Improved summarization for large pull requests

## Mid Term

- Additional providers behind the same adapter interface
- Repository context fetches for related files when diff-only review is insufficient
- Optional SARIF or Check Run output for richer CI integrations

## Longer Term

- Fix suggestions with guarded opt-in behavior
- Historical issue learning for team-specific review preferences
- Review analytics for false positive tracking and prompt tuning
